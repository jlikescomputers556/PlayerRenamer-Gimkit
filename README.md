# PlayerRenamer
PlayerRenamer is a simple script used with gimloader in gimkit to change the names of players on your screen. Sadly it won't change on everyone's screens, but it works as it is for now. 
---
## Install
* You need to have a [Gimkit](gimkit.com) account.
* Download [Gimloader](https://gimloader.github.io/) as a chrome extension.
* Create a new plugin from the plugins menu on Gimkit.
